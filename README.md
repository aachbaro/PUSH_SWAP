## push_swap (42)

Projet consistant à trier une liste d’entiers en utilisant un ensemble limité d’opérations sur deux piles.

### Contenu

- Implémentation d’algorithmes de tri
- Optimisation du nombre d’opérations
- Manipulation de structures de données (piles)

### Contexte

Projet de l’école 42 visant à développer la réflexion algorithmique, l’optimisation et la compréhension des structures de données en C.
